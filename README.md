# Vernity Web - Portfólio React

Um site de portfólio moderno e elegante desenvolvido em React, com design responsivo e animações suaves usando Framer Motion.

## 🚀 Características

- **React 18**: Framework moderno com hooks e componentes funcionais
- **Framer Motion**: Animações suaves e profissionais
- **Design Moderno**: Interface escura e elegante inspirada em tendências atuais
- **Totalmente Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Componentes Reutilizáveis**: Arquitetura modular e limpa
- **Performance Otimizada**: Carregamento rápido e SEO-friendly
- **Formulário Funcional**: Validação e feedback visual

## 🛠️ Stack Tecnológica

- **React 18.2.0**: Framework principal
- **Framer Motion**: Animações e transições
- **React Intersection Observer**: Detecção de scroll
- **CSS3**: Estilos modernos com Flexbox e Grid
- **JavaScript ES6+**: Funcionalidades interativas
*Google Fonts**: Tipografia Inter

## 📱 Seções

1. **Hero Section**: Banner principal com sua frase de impacto
2. **Sobre**: Informações pessoais e habilidades técnicas
3. **Serviços**: Cards com seus principais serviços
4. **Portfólio**: Grid de projetos com hover effects
5. **Contato**: Formulário funcional e informações de contato
6. **Footer**: Links rápidos e redes sociais

## 🚀 Como Executar

### Pré-requisitos
- Node.js (versão 14 ou superior)
- npm ou yarn

### Instalação

1. **Clone o repositório**
```bash
git clone <url-do-repositorio>
cd vernity-web-portfolio
```

2. **Instale as dependências**
```bash
npm install
```

3. **Execute o projeto**
```bash
npm start
```

4. **Abra no navegador**
```
http://localhost:3000
```

### Build para Produção
```bash
npm run build
```

## 🎯 Personalização

### Alterando o Conteúdo

1. **Título e Navegação**: Edite o nome "Vernity Web" nos componentes
2. **Frase Principal**: Modifique o texto no componente `Hero.js`
3. **Informações Pessoais**: Atualize a seção no componente `About.js`
4. **Serviços**: Edite o array de serviços no componente `Services.js`
5. **Portfólio**: Modifique o array de projetos no componente `Portfolio.js`
6. **Contato**: Atualize as informações no componente `Contact.js`

### Alterando Cores

No arquivo `App.css`, você pode modificar:
- `#ff4444` - Cor de destaque (vermelho)
- `#0a0a0a` - Fundo principal
- `#111111` - Fundo das seções
- `#ffffff` - Texto branco

### Adicionando Sua Foto

1. Substitua o arquivo `banner.webp` na pasta `public/images/favicon_io (11)/`
2. A imagem será exibida automaticamente no banner
3. Ajuste o tamanho conforme necessário no CSS

### Pasta de Imagens

Arquivos de imagem do projeto:

- `banner.webp` - Sua imagem do banner (em `public/images/favicon_io (11)/`)
- `logo.svg` - Logo da Vernity Web
- `favicon.ico` - Favicon do site (na pasta public/)

## Formatos utilizados:
- WebP (banner - melhor compressão e qualidade)
- SVG (logo - vetorial, escalável)
- ICO (favicon)

## Vantagens do WebP:
- Compressão superior ao JPEG/PNG
- Qualidade visual excelente
- Carregamento mais rápido
- Suporte moderno em todos os navegadores
