# Imagens Necessárias

Para que o site funcione corretamente, você precisa adicionar as seguintes imagens nesta pasta:

## Arquivos Obrigatórios:

1. **`logo.svg`** - Logo da Vernity Web (formato SVG recomendado)
2. **`banner.webp`** - Imagem principal do banner (formato WebP recomendado)
3. **`favicon.ico`** - Favicon do site (48x48px)
4. **`favicon-large.ico`** - Favicon maior (270x270px para Apple Touch)

## Imagens do Portfólio:

5. **`dommaelectrasite.webp`** - Preview do site Domma Electra
6. **`crispsicologosite.webp`** - Preview do site Cris Psicólogo

## Formatos Recomendados:

- **SVG** para logos e ícones
- **WebP** para imagens de conteúdo (melhor performance)
- **ICO** para favicons

## Tamanhos Sugeridos:

- **Logo**: 120x40px (SVG)
- **Banner**: 800x600px (WebP)
- **Previews do portfólio**: 800x600px (WebP)
- **Favicon**: 48x48px e 270x270px (ICO)

## Como Adicionar:

1. Coloque as imagens nesta pasta (`public/images/`)
2. Certifique-se de que os nomes dos arquivos estão exatamente como listados acima
3. Use formatos otimizados para web (WebP, SVG, ICO)

## Otimização:

- Comprima as imagens antes de adicionar
- Use WebP para melhor performance
- Mantenha os arquivos menores que 500KB cada
