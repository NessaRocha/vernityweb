# Pasta de Imagens

Arquivos de imagem do projeto:

- `banner.svg` - Sua imagem do banner
- `logo.svg` - Logo da Vernity Web
- `favicon.ico` - Favicon do site (na pasta public/)

## Formatos utilizados:
- SVG (vetorial, melhor qualidade e performance)
- ICO (favicon)

## Vantagens do SVG:
- Escalável sem perda de qualidade
- Menor tamanho de arquivo
- Melhor performance
- Suporte a cores e animações
